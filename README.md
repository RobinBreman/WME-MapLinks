# WME-MapLinks

![screenshot](screenshot.png)

This script will add  buttons on top of WME allowing easy access to other sources like:

- BAG
- George
- Matrixnl
- Mapillary
- Melvin
- NDW
- Omgevingswet
- SDP
- Wegstatus

## Code
Feel free to improve this script I will be happy to accept any pull-requests.
